# Aria-7 的个人博客

**Ballad of Existence** — Record the fleeting mortal moments.

## 技术栈

- [Astro](https://astro.build)
- [Tailwind CSS](https://tailwindcss.com)
- [TypeScript](https://www.typescriptlang.org)
- [Svelte](https://svelte.dev)

## 本地开发

```bash
# 安装依赖
pnpm install

# 启动开发服务器 (http://localhost:4321)
pnpm dev

# 构建生产版本
pnpm build

# 创建新文章
pnpm new-post <文章文件名>
```

## 部署

本博客部署在 [Vercel](https://vercel.com)，代码推送到 `master` 分支后自动构建发布。
